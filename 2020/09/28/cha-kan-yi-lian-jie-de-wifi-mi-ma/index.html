查看已连接的WiFi密码

密码 WIFI
技巧
发布日期:   2020-09-28
更新日期:   2020-11-18
文章字数:   734
阅读时长:   2 分
阅读次数:  

问题描述

​ 有将WLAN设置为在信号范围内自动连接的习惯，今天想在一个新设备上连接WiFi时发现忘记密码了。(:-……

解决思路

​ 现在有一台电脑和一部手机已经连接到WiFi上，在不直接去路由器中查看WiFi密码或使用WiFi破解工具的情况下，想通过查看已连接设备中存储的WiFi密码来得到WiFi密码。

解决步骤

电脑端

图形界面

环境：

  • 系统：Windows10
  • 热点：CMCC-335574918

步骤：

  1. 连接SSID为CMCC-335574918的热点，然后点击【网路和Internet设置】，打开网络和Internet设置界面。

  2. 点击【网络和共享中心】，打开网络和共享中心设置界面。

  3. 点击已连接的WLAN(CMCC-335574918)后显示WLAN状态界面。

  4. 点击【无线属性按钮】后显示CMCC-335574918无线网络属性界面。

  5. 点击【安全】选项卡后显示无线网络安全对应信息。

  6. 勾选显示字符选项，即可查看CMCC-335574918无线网络的密码。

局限性：

  • 必须是正在连接的网络

命令行界面

  1. 使用管理员身份打开PowerShell，【Win+X】 → 选择【Windows PowerShell（管理员）】选项，操作结果如下图所示。

  2. 使用netsh wlan show profile命令查看当前主机连接过程的所有WiFi的配置文件名称，操作结果如下图所示。

  3. 使用netsh wlan show profile CMCC-335574918 key=clear命令查看指定WiFi配置文件信息，操作结果如下图所示。

手机端

环境：

  • 系统：Android
  • 热点：CMCC-EDU

步骤：

  1. 确保无线已连接成功。

  2. 点击手机的设置图标，进入设置界面。

  3. 点击无线网络选项后进入无线网络界面。

  4. 找到已连接的WiFi并点击详情图标。

  5. 点击界面中的分享网络，接着会生成一个二维码，截图保存。

  6. 使用微信扫一扫功能，扫描截取的二维码，得到无线密码信息。

问题反思

  • 人的记忆存在一点的有效期，某些记忆在一定时间后会被遗忘。因此我们可以选好记或印象深刻的信息作为密码。
  • 在设置密码时不要设置简单常用的密码，防止密码被爆破成功。
  • 对于种类繁多的密码，我们可以密码管理工具帮助我们整理和存储密码。
  • 已连接的WiFi密码会存储在设备上，我们可以通过简单的操作得到已连接WiFi的密码。
  • 对于一个未知密码的热点，我们可以使用WiFi万能钥匙或其他工具先连接上该热点，再通过扫描分享网络的二维码得到该热点的密码。
