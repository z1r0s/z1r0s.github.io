域和活动目录

域概念

什么是域

​ 逻辑上把网络中计算机组织在一起，集中化管理的一种环境。

  • 域是活动目录的一种实现形式。
  • 域是活动目录中最核心的管理单位
  • 域由域控制器和成员计算机组成

什么是活动目录

  • 是一个数据库：存储着整个Windows域中用户、组、计算机、共享文件夹等对象信息
  • 是一种服务：能实现对数据对象的增、删、改、查等操作

域包含的成员

  • 域控制器（DC）：安装了活动目录的一台计算机

    • 域控制器 → 成员服务器
    • 域控制器 → 独立服务器
    • 成员服务器 → 域控制器
    • 独立服务器 → 域控制器

  • 成员计算机：加入了该域环境的计算机

    • 成员计算机 → 独立计算机
    • 独立计算机 → 成员计算机

  • 成员服务器：加入了该域环境的服务器

    • 成员服务器 → 域控制器
    • 成员服务器 → 独立服务器
    • 域控制器 → 成员服务器
    • 独立服务器 → 成员服务器

  • 独立服务器：工作组环境的服务器，没有加入到域

    • 独立服务器 → 域控制器
    • 独立服务器 → 成员服务器
    • 成员服务器 → 独立服务器
    • 域控制器 → 成员服务器 → 独立服务器

      域的优点

  • 集中管理

  • 便捷的网络资源访问

  • 可扩展性

域的结构

  • 单域：网络中只有一个域 （最常见的管理方式）
  • 域树：具有连续名称空间的多个域
  • 域林：由一个或多个没有形成连续名称的域树组成

组策略

​ 一组策略的集合，用来用来统一修改系统、设置程序（设置桌面环境、安全设置、自动执行脚本、软件分发）

组策略优点

  1. 减小管理成本：只需设置一次，相应的计算机或用户即可应用
  2. 减小用户单独配置错误的可能性
  3. 可以针对特定对象设置特定的策略（用户策略、计算机策略）

组策略对象（Group Policy Object）

​ 存储组策略的所有配置信息，AD中的一种特殊对象。

默认GPO

  • 默认域策略:影响域中所有的用户和计算机

  • 默认域控制器策略:影响组织单位“Domain Controllers”中所有的用户和计算机

组策略配置分类

  • 计算机配置
    • 只针对容器中的计算机生效
    • 策略：
      • 软件设置
      • Windows设置
      • 管理模板
    • 首选项
      • Windows设置
      • 控制面板设置
    • 配置完成重启客户端生效
  • 用户配置
    • 只针对容器中的用户生效
    • 策略：
      • 软件设置
      • Windows设置
      • 管理模板那
    • 首选项：
      • Windows设置
      • 控制面板设置
    • 配置完成重启客户端生效

组策略应用规则

  • 策略继承：下级容器默认会继承来自上级容器的GPO
  • 策略阻止：子容器可以阻止继承上级容器的GPO
  • 策略累加：如果多个组策略设置不冲突，则最终的有效策略是所有组策略设置的累加
  • 策略冲突：如果多个组策略设置冲突，则后应用的组策略覆盖先应用的组策略
  • 策略强制生效：上级容器强制下级容器执行其GPO设置，“强制生效”会覆盖“阻止继承”设置

组策略应用顺序

组策略应用顺序：LSDOU

  • 首先应用本地组策略，如果有站点组策略，则应用
  • 接着应用域策略
  • 最后应用OU上的策略
  • 如果同一个OU上链接了多个GPO，则按照链接顺序从高到低逐个应用

域控制器的创建

先决条件

  • 安装者必须是本地管理员权限
  • 操作系统版本必须是服务器版本（Web版除外）
  • 本地至少一个分区是NTFS文件格式
  • 配置静态IP和子网掩码
  • 磁盘空间足够

额外域控制器

为什么要安装额外域控

  • 容错：当域中的域控制器出现故障时，其他额外域控制器仍能提供域服务.
  • 负载均衡：适用大型网络环境，加快用户登录域的审核速度.
  • 更易于用户的连接：适用于分支机构，提高登录速度.

安装额外域控制器

先决条件

  • 必须具有域管理员身份
  • 正确的TCP/IP设置 （能保证和域控制通信，DNS设置正确）
  • 操作系统版本必须受当前域功能级别支持 （向下兼容）

注意事项:

  • 安装过程中不需要加入域也能成功
  • 安装完成后，AD中所有数据是同步的

部署成员计算机

先决条件

  • 客户机必须和域控制器是连通的
  • 一定要配置正确的DNS地址 （否则无法解析域名，也就无法加入域）

域的管理

功能级别提升（域、林）

  • Windows 2000纯模式
  • Windows Server 2003
  • Windows Server 2008
  • Windows Server 2008 R2
    林功能级别限制域功能级别,域功能级别限制域控制器的操作系统
    域控制器的操作系统<=域功能级别<=林功能级别

用户管理

命令规则：

  • 唯一的用户登录名 （登录名在域中是唯一的；显示名在OU中唯一）
  • 最长20字符
  • 不能出现非法字符 （< > ? ! / \ | = 等）

OU管理

​ 是活动目录中的一个容器（组织单位）；可以存放用户、组、计算机、OU等其他对象

创建原则：

  • 基于部门的
  • 基于地理位置的
  • 基于对象类型的
    特别注意：OU中不能包含来自其他域中的对象

组管理

组类型：

  • 安全组：主要是用于设置用户安全权限的
  • 通讯组：主要用于电子邮件通信的

组作用域：

  • 本地域组：使用范围是本域
  • 全局组：使用范围是整个林及信任域
  • 通用组：使用范围是整个林及信任域 （用户的登录或查询速度更快）
